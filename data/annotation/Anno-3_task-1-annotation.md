## Instructions

For each reasoning step, distribute exactly **100 points** across the five ethical frameworks based on how strongly each is invoked.

1. **Kantian Deontology**: Duty-based reasoning (duties, rules, obligations, rights, universal principles)
2. **Benthamite Act Utilitarianism**: Outcome-focused (consequences, harms, benefits, welfare)

4. **Scanlonian Contractualism**: Relationship-focused (interpersonal justifiability, mutual agreement, care)
5. **Gauthierian Contractarianism**: Fairness-focused (rational agreement, mutual advantage, social cooperation)

**Scoring Rules:**
- Scores MUST sum to exactly 100
- 0 = framework completely absent
- Consider both explicit and implicit invocations
